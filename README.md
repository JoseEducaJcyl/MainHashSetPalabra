# Eliminador de Palabras Duplicadas

## Descripción
Aplicación interactiva en Java que permite al usuario introducir palabras y demuestra cómo HashSet elimina automáticamente los duplicados, mostrando solo las palabras únicas al final.

## Funcionalidades
- Solicitar palabras al usuario hasta que escriba "fin"
- Almacenar todas las palabras introducidas en un ArrayList
- Convertir el ArrayList a HashSet para eliminar duplicados automáticamente
- Mostrar solo las palabras únicas (sin repeticiones)

## Tecnologías
- Java
- ArrayList
- HashSet
- Scanner
- Bucle do-while
